# 🧠 OS Page Replacement Simulator

A modern GUI-based Operating Systems project that simulates and compares page replacement algorithms with real-time visualization.

---

## 🚀 Features
- FIFO, LRU, LFU, Optimal algorithms
- Step-by-step frame animation
- Hit 🟢 and Fault 🔴 visualization
- Page replacement tracking
- Speed control for animation
- Graph comparison using Matplotlib
- Modern UI using CustomTkinter

---

## 📊 Algorithms Implemented
- First In First Out (FIFO)
- Least Recently Used (LRU)
- Least Frequently Used (LFU)
- Optimal Page Replacement

---
